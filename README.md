Implementação em JavaScript moderno dos exercícios do livro **"Piensa en Haskell"**.

Este projeto traduz os conceitos de programação funcional de Haskell para JavaScript (ES2022+), com código puro, documentado com JSDoc e coberto por testes.

## Sobre o material original

**Piensa en Haskell** é uma obra de **José A. Alonso Jiménez**, catedrático da Universidade de Sevilha, disponibilizada gratuitamente para a comunidade.

- Site oficial: https://jaalonso.github.io/cursos/pih/
- Repositório original: https://github.com/jaalonso/PIH

Todos os enunciados, exemplos e a estrutura pedagógica pertencem ao autor original e são utilizados aqui exclusivamente para fins educacionais.

## Instalação

    git clone https://github.com/lukkaslt/haskell-exercises-js.git
    cd haskell-exercises-js
    npm install

## Uso

    npm test

Importe as funções diretamente:

    import { divisaoSegura } from './src/basico/index.js'
